# Capas dos produtos (opcional — pasta local)

**Recomendado:** gerenciar produtos em **Admin → Produtos (Biblioteca)**  
(`/admin/catalogo/produtos`) — título, descrição, capa (upload Cloudinary) e link Hotmart.

Esta pasta serve para capas locais (`/catalog/arquivo.jpg`) se preferir não usar URL do Cloudinary.

| Arquivo sugerido | Uso |
|------------------|-----|
| `livro-o-corpo-diz.jpg` | Livro digital |
| `pdf-100-sintomas.jpg` | PDF |
| `audioterapia.jpg` | Audioterapia |
| `placeholder.svg` | Fallback automático |
